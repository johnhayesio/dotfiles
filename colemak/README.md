# How to keyboard

1. Copy Colemak Programmer.bundle to ```/Library/Keyboard\ Layouts```

   #### Note: Do not copy to ```~/Library/Keyboard\ Layouts```

2. Restart MacOSX.

3. Pick keyboard layout by selecting other input sources from login screen.
